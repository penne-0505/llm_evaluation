## タスク: 豚が穢れている理由（宗教規範の起源）
## task_type: speculative
## weights: logic_and_fact=40, constraint_adherence=20, helpfulness_and_creativity=40

## 評価目的

ユダヤ教・イスラム教圏の豚忌避を、教義の引用で止めず、環境・生業・経済、象徴分類、
集団境界、制度化と継承という複数の層から説明し、単一原因を確定事実として断定せずに
「なぜ規範になり、なぜ残ったか」を考察できるかを評価する。

## 採点者向け事実情報

- 豚タブーの起源について、学術的に単一の確定原因があるわけではない。
- 文化唯物論的説明は、乾燥化した環境や牧畜・農耕体系における飼育コスト、資源競合、
  豚から得られる便益の構成などから禁忌形成を説明する。有力な仮説の一つだが、地域差や
  考古学的反例をすべて説明する唯一説ではない。
- 衛生・寄生虫説は直感的だが、他の肉にも危険があること、十分な加熱で対処できること、
  古代人の因果認識を直接示しにくいことから、単独の起源説明としては不足する。
- 象徴的分類、宗教共同体の identity、隣接集団との差異化、規範が制度化された後の
  path dependence は、禁忌の意味と持続を説明し得る。
- 「最初に不利だった理由」と「聖典へ書かれた理由」と「後代まで守られた理由」は同じとは限らない。

なし。

## 評価ルーブリック

### 1. Logic & Fact（0〜40点）

#### 評価観点

- 環境・経済条件から食習慣、社会規範、宗教的 codification へ至る因果の橋を示しているか。
- 起源、制度化、持続を時間的に区別しているか。
- materialist、symbolic、identity、health などの仮説を、証拠と限界を含めて扱うか。
- 「中東は暑いから腐る」などの単純な決定論へ還元していないか。

#### 得点アンカー

| 得点帯 | 基準 |
| --- | --- |
| 36〜40 | 少なくとも二つの説明層を因果的に接続し、起源・制度化・持続の違い、各仮説の適用範囲と限界を明確にする。単一原因を装わない。 |
| 26〜35 | 複数層と主要な留保はあるが、因果の橋、時系列、証拠評価のいずれかが弱い。 |
| 12〜25 | 環境説、衛生説、identity 説などを挙げるが、列挙的または一説への過度な依存がある。 |
| 0〜11 | 教義の再述だけ、単純な衛生決定論、明確な歴史的誤認、または問いと無関係。 |

### 2. Constraint Adherence（0〜20点）

#### 評価観点

- 「宗教的に決まっているから」の一段前を知りたいという要求へ直接答えているか。
- ユダヤ教とイスラム教を完全に同一の成立過程として雑にまとめず、必要な時間差・継承関係を示すか。
- 仮説を確定事実として売り込まず、どこまでが推論かを明示しているか。

#### 得点アンカー

| 得点帯 | 基準 |
| --- | --- |
| 18〜20 | 教義の背景構造を中心に置き、複数宗教と時代の関係を過度に単純化せず、推論の確度を適切に示す。 |
| 13〜17 | 起源には答えるが、教義説明との混在、宗教間の一括り、または確度表現に小さな粗さがある。 |
| 6〜12 | 教義紹介が中心で、背景仮説は短い。あるいは一つの仮説を「理由」と断定する。 |
| 0〜5 | ユーザーが退けた説明を繰り返し、起源へほぼ踏み込まない。 |

### 3. Helpfulness & Creativity（0〜40点）

#### 評価観点

- 複数仮説を、競合する説の羅列ではなく「何を説明する説か」で比較しているか。
- 具体的な反例、地域差、検証に必要な証拠を示し、ユーザーが単純説を見分けられるか。
- 結論を曖昧にするだけでなく、現時点で妥当な総合像を提示しているか。

#### 得点アンカー

| 得点帯 | 基準 |
| --- | --- |
| 36〜40 | 各説を起源・意味・持続などの説明対象で整理し、強みと限界を比較した上で、多因子的な総合像を分かりやすく提示する。 |
| 26〜35 | 複数説と比較は有用だが、具体例、反例、総合のいずれかが弱い。 |
| 12〜25 | 情報はあるが列挙的で、どの説が何を説明するか、どう組み合わせるかが分かりにくい。 |
| 0〜11 | 単一説の断定、教科書的な宗教紹介、またはユーザーの疑問を深める材料がほぼない。 |

## 軸間の切り分け

- 歴史・人類学的な因果と仮説の妥当性は `logic_and_fact` で評価する。
- 教義の背後を問う要求、宗教間の区別、確度の明示は `constraint_adherence` で評価する。
- 仮説比較、反例、総合像の分かりやすさは `helpfulness_and_creativity` で評価する。
- 単一原因への還元を三軸で反復せず、事実・推論上の粗さ、問いへの不適合、比較不能という別の影響に分ける。
