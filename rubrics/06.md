## タスク: 清少納言風の描写
## task_type: creative
## weights: logic_and_fact=30, constraint_adherence=30, helpfulness_and_creativity=40

## 評価目的

淹れたての少し安いインスタントコーヒーを、古語の置換だけでなく、具体を素早く観察し、
好悪を小気味よく裁き、欠点の中にも趣を見出す『枕草子』を想起させる感性で描けるかを
評価する。特定の古語、定型句、原文模倣は必須ではない。

## Critical Fail Conditions

なし。

## 評価ルーブリック

### 1. Logic & Fact（0〜30点）

#### 評価観点

- 清少納言を想起させる観察の細かさ、列挙、対比、断定のリズム、批評的な視線があるか。
- 古語または古典調を用いる場合、読解を壊すほど文法・語義が崩れていないか。
- 現代物を古典の視座へ移すための一貫した語り手がいるか。
- 単に「雅」「あはれ」「をかし」などを散らすだけになっていないか。

#### 得点アンカー

| 得点帯 | 基準 |
| --- | --- |
| 27〜30 | 具体観察、短い価値判断、対比や列挙のリズムが自然に働き、語り手の視座が一貫して『枕草子』を想起させる。現代語主体でもよい。 |
| 20〜26 | 古典的なリズムや視線は認められるが、一部が表層的な古語化、説明調、または不自然な文法になる。 |
| 9〜19 | 古語や定型句はあるが、観察と批評の運動が弱く、現代文の語尾置換に近い。 |
| 0〜8 | 指定作家・作品を想起させる視座がほぼなく、一般的な感想文または無関係な文体。 |

### 2. Constraint Adherence（0〜30点）

#### 評価観点

- 「淹れたて」「少し安い」「インスタントコーヒー」という三つの特徴が描写へ現れているか。
- 安さを単純に侮蔑するだけでも、上質な本格コーヒーとして美化するだけでもなく、対象固有の可笑しみや愛着を捉えているか。
- 説明・分析ではなく、依頼された創作として成立しているか。

#### 得点アンカー

| 得点帯 | 基準 |
| --- | --- |
| 27〜30 | 対象の熱、香り、粉っぽさ、手軽さ、器や場面などから、安さと淹れたての魅力を複層的に描く。三つの指定が自然に統合される。 |
| 20〜26 | 指定対象は明確だが、安さ、淹れたて、インスタント性の一つが弱い、または評価が一方向に偏る。 |
| 9〜19 | コーヒー一般の描写へ寄り、指定された「少し安い」対象の個性が薄い。あるいは解説が創作を圧迫する。 |
| 0〜8 | 対象を取り違える、指定要素をほぼ描かない、または創作を行わない。 |

### 3. Helpfulness & Creativity（0〜40点）

#### 評価観点

- 現代の生活用品と古典的感性の衝突から、固有の比喩、人物観察、場面の可笑しみを生んでいるか。
- 音読したときの調子、文の長短、余韻が読み手を楽しませるか。
- 長さにかかわらず、一篇として焦点と終わり方があるか。

#### 得点アンカー

| 得点帯 | 基準 |
| --- | --- |
| 36〜40 | 対象固有の発見があり、古典的な観察眼と現代生活のずれが自然な可笑しみや余韻を生む。表現が鮮明で一篇としてまとまる。 |
| 26〜35 | 読ませる比喩や場面はあるが、発見の独自性、リズム、全体のまとまりのいずれかが弱い。 |
| 12〜25 | 定型的な古文調としては成立するが、対象固有の面白さや表現の変化が少ない。 |
| 0〜11 | 単純な好悪、陳腐な古語の羅列、または読み手を楽しませる描写がほぼない。 |

## 軸間の切り分け

- 清少納言を想起させる視座・リズム・語用の一貫性は `logic_and_fact` で評価する。
- 指定対象と創作形式への忠実さは `constraint_adherence` で評価する。
- 比喩、場面、可笑しみ、読後感の独自性は `helpfulness_and_creativity` で評価する。
- 「古典らしくない」という一つの印象を全軸で反復せず、文体理解、対象逸脱、表現価値を分けて判断する。
