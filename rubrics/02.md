## タスク: 歯笛のメカニズムと練習
## task_type: fact
## weights: logic_and_fact=60, constraint_adherence=30, helpfulness_and_creativity=10

## 評価目的

既に歯笛を吹ける上級者に対し、確立した教本が乏しいことを踏まえ、発音を単一のコツへ
還元せず、気流による音源と口腔形状による共鳴の結合系として仮説化し、その仮説から
再現可能な練習実験を導けるかを評価する。

## 採点者向け事実情報

- 歯笛そのものを対象にした直接的な音響研究は乏しい。したがって、音源の正確な位置や
  単一の発音モデルを既知の定説として断定せず、観察可能な予測を持つ作業仮説として扱う。
- 妥当な作業仮説の一つは、歯またはその近傍の狭窄を通る jet が作る空力的な音源と、口腔内の
  共鳴が相互作用するというモデルである。ただし、具体的な歯列・舌形・発音様式によって、
  支配的な音源や共鳴条件が異なる可能性がある。
- 通常の唇口笛に関する研究は、jet と声道共鳴、舌位置と pitch の関係を考える参考にはなるが、
  歯笛と同一の音源形状を立証するものではない。類推と同一視を区別する。
- pitch、立ち上がり、音量、音色は、舌・顎・唇・歯の近傍の開口、口腔容積、jet の速度と
  方向などに連成して依存する。単純に「口全体を小さくすれば高音／低音」のような一変数の
  法則へ断定しない。
- 口腔容積と外への狭い開口は別の変数である。「口腔を広く保ちながら局所的な開口を狭める」
  ことは矛盾しない。
- 個人差が大きく、歯列や既得技術によって最適形状は異なるため、機構説明には不確実性の校正が必要である。

## Critical Fail Conditions

| ID | 条件 |
| --- | --- |
| CF-1 | 声帯振動を歯笛の主要な音源として説明し、その前提から練習法を組み立てている。 |
| CF-2 | 指笛または唇で作る通常の口笛と同一の発音機構であると断定し、歯の近傍の気流・狭窄を扱わないまま、その技法を中心策として転用している。 |

用語の揺れ、通常の口笛からの明示的な類推、複数仮説の比較だけでは CF にしない。最終的な
機構モデルがどこに置かれ、歯笛固有の観察で更新できる形になっているかで判断する。

## 評価ルーブリック

### 1. Logic & Fact（0〜60点）

#### 評価観点

- 空力的音源と共鳴の双方、またはそれらに代わる首尾一貫したモデルを、確度に応じた言葉で説明しているか。
- 舌、顎、局所開口、口腔容積、息の速度・方向などを混同せず、操作と予想される変化を結び付けているか。
- 直接証拠の乏しさを踏まえ、単純な決め打ちを避け、観察で仮説を比較・更新できる説明になっているか。

#### 得点アンカー

| 得点帯 | 基準 |
| --- | --- |
| 54〜60 | 直接証拠と類推を区別し、音源・共鳴・jet などの作業仮説、または同等に整合的な代替モデルから、各操作が pitch、音色、安定性へどう影響し得るかを検証可能な予測として導く。個人差と反証条件も扱う。 |
| 39〜53 | 機構の主要部分は妥当で練習へ接続しているが、一部を定説化する、変数を単純化しすぎる、または代替仮説・不確実性の扱いが弱い。 |
| 18〜38 | エッジ、共鳴、息など正しい要素はあるが、相互関係が曖昧で一般的な口笛の説明と十分に区別できない。 |
| 0〜17 | 中心機構が誤っている、説明と練習が接続しない、または感覚論だけで機構考察がほぼない。 |

### 2. Constraint Adherence（0〜30点）

#### 評価観点

- ユーザーが既に上手く吹けることを前提に、初心者向けの発音成功だけへ退行していないか。
- ネット上の定説を装わず、基本原理からの推論と不確実な部分を区別しているか。
- 「メカニズムに基づく最適なアプローチ」という要求へ直接応えているか。

#### 得点アンカー

| 得点帯 | 基準 |
| --- | --- |
| 27〜30 | 上級者向けの改善変数を選び、既知事項、他の口笛からの類推、自分の作業仮説、観測可能な予測を明確に分けている。 |
| 20〜26 | 要求には概ね応えるが、初心者向け説明が長い、または経験則と推論の境界が一部曖昧。 |
| 9〜19 | 一般的な「練習あるのみ」や通常の口笛 tips が中心で、機構からの導出が弱い。 |
| 0〜8 | 技量前提と方法要求をほぼ無視し、別技法または精神論へ置き換えている。 |

### 3. Helpfulness & Creativity（0〜10点）

#### 評価観点

- 一度に一変数を変える、録音・tuner・spectrogram 等で比較する、再現条件を記録するなど、
  仮説を検証できる練習設計があるか。特定ツールの使用自体は必須ではない。
- pitch range、立ち上がり、音量、持続、音色など、改善目標を分けて試せるか。
- 痛み、顎の過緊張、めまいなどが出た場合に中止する安全配慮があるか。

#### 得点アンカー

| 得点帯 | 基準 |
| --- | --- |
| 9〜10 | 上級者がすぐ実施できる比較実験、記録方法、成功指標があり、安全に反復できる。 |
| 7〜8 | 具体的な drill はあるが、変数分離、測定、記録のいずれかが弱い。 |
| 3〜6 | 動作提案はあるが、結果の見分け方がなく試行錯誤が漫然としている。 |
| 0〜2 | 抽象的助言だけ、または危険な過緊張・過換気を推奨する。 |

## 軸間の切り分け

- 発音機構と操作予測の妥当性は `logic_and_fact` で評価する。
- 上級者前提、原理からの推論、不確実性の明示は `constraint_adherence` で評価する。
- drill、測定、記録、安全な反復の設計は `helpfulness_and_creativity` で評価する。
- ある練習法が誤った機構に基づく場合、機構誤りを Logic で主に減点し、同じ誤りを単に「役立たない」と言い換えて重複減点しない。
