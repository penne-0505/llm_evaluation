## タスク: 文体・言語運用の包括評価
## task_type: holistic
## weights: logic_and_fact=40, constraint_adherence=30, helpfulness_and_creativity=30

## Critical Fail Conditions（禁忌回答）

以下のいずれかに該当する場合、`"critical_fail": true` としてください。
該当する場合、各スコアの採点は行わず、すべて0点としてください。

| ID | 条件 |
|----|------|
| CF-1 | **翻訳調の固定化**: 「〜することができます」「〜という形になります」「〜に関しては」など、機械翻訳・マニュアル調の紋切り型表現が、複数の出力にわたって反復的・支配的に現れており、全体的な読みにくさを引き起こしている |
| CF-2 | **語義誤用の反復**: 特定の語や表現を、本来とは異なる意味・用法で一貫して使っているパターンが、2つ以上の出力で確認できる（単発の言い間違いではなく、体系的な誤用として） |

## 評価ルーブリック

### 1. Logic & Fact（0〜40点）

語の選択と語用の正確さを評価します。文脈に対して最もニュアンスが合う語を選べているか、特定の表現への過依存がないかを見てください。

> **採点上の注意**: 「より良い言い回しがある」程度の改善余地は減点対象としない。明確に不適切・不正確、または全体の読みやすさを損なうレベルの語選択の問題を対象とする。
>
> **外国語・異体字の混入について**: 意図的なコードスイッチング（例：日本語に相当語のない専門用語をそのまま使う）はニュアンスへの貢献として許容する。一方、対象言語に存在しない文字体系の混入（例：簡体字が日本語テキストに漏れ出す）は、著者の選択ではなくモデルの言語漏洩とみなし、Axis 1の減点事由として扱う。判断基準は「その混入が文意・ニュアンスに貢献しているか」。

| 点数 | 基準 |
|------|------|
| 40点 | 複数のタスクにわたって語の選択が一貫して適切。文脈に対して精度の高い語が選ばれており、表現パターンへの過依存も見られない |
| 28点 | 全体的には適切だが、一部のタスクで「より正確な語があった」程度の選択の甘さがある。または特定の表現（例：「〜という形で」）に軽度の依存傾向がある |
| 15点 | 語選択の問題が複数のタスクにまたがっており、文意が若干ずれる場面や、定型表現への明確な依存が確認できる |
| 0点  | 語義を誤用している、または紋切り型表現が支配的で、全体の語の質が低いと判断せざるを得ない |

---

### 2. Constraint Adherence（0〜30点）

文体の一貫性と文脈適合を評価します。各タスクの性格（格式・くだけ度・トーン）に合った文体を選べているか、また複数の出力を通じて一貫した「声」があるかを見てください。

> **採点上の注意**: タスクの種類が異なれば文体が変化するのは自然であり、それ自体は減点しない。評価するのは「各タスクの文脈に対して文体が適切かどうか」と「不必要な揺れや唐突な不整合がないかどうか」。

| 点数 | 基準 |
|------|------|
| 30点 | 各タスクに対して適切な文体・トーンを選べており、かつ複数の出力を通じて一貫した「声」がある。不整合や唐突な揺れが見当たらない |
| 20点 | おおむね適切だが、一部のタスクで文体が若干の過剰・不足（堅すぎる・くだけすぎ）、または出力間で小さな不整合がある |
| 10点 | 文体の選択に明確なミスマッチが1〜2箇所ある（例：技術的な質問に対してくだけた文体、雑談に対して過度に格式ばった文体）、または出力間の揺れが気になるレベルで存在する |
| 0点  | 文体が文脈とほぼ合致していない、または複数の出力で文体が統一されていない |

---

### 3. Helpfulness & Creativity（0〜30点）

読みやすさと表現の質を評価します。文の構造の明快さ、冗長さのなさ、表現の変化と自然さを見てください。

> **採点上の注意**: 文を短くすれば良いというわけではない。内容に見合った長さで、過不足なく伝えられているかを重視する。

| 点数 | 基準 |
|------|------|
| 30点 | 文構造が明快で冗長さがなく、日本語として自然に読める。定型表現に頼らず、文ごとに適切な表現の変化があり、全体を通じて読みやすい |
| 20点 | 概ね読みやすいが、一部に過度な入れ子・冗長な説明・同一表現の繰り返しがある |
| 10点 | 冗長さや定型表現への依存が複数の出力で確認でき、読み進めるのに若干の負荷がある。または文構造が複雑すぎる箇所が散見される |
| 0点  | 冗長・不明瞭・マニュアル調の記述が支配的で、全体を通じた読みやすさが著しく損なわれている |
